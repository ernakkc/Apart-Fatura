# Apart Fatura

Bu proje, apartmanlar için kolayca fatura oluşturmanıza olanak tanıyan bir masaüstü uygulamasıdır.

## Özellikler
- Modern ve kullanıcı dostu arayüz (PyQt5)
- PDF fatura oluşturma ve düzenleme
- Fatura şablonu ve ikon desteği

## Kurulum
1. Python 3.8+ kurulu olmalı.
2. Gerekli paketleri yükleyin:
   ```bash
   pip install -r requirements.txt
   ```
3. Uygulamayı başlatın:
   ```bash
   python main.py
   ```
   
```_Pair Extraordinaire_```

## Gereksinimler
- PyQt5
- reportlab
- PyPDF2

## Kullanım
Formu doldurun ve faturanızı PDF olarak oluşturun.
